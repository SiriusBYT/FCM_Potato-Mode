Flashcord

A collection of variables to make Flashcord much less laggy on extremely bad computers. Not to be confused with Flashcord's LPM branch.

This module features the following changes:

- Quadrupled Animation / Transition Speed

- Switches the GuildNav from Vertical to Horizontal to accomodate from the lost chat space from disabling ChatSlide and MemberSlide

- Sets Chat Effects to Text Mode

- Disables Guild Notification and Channel Notification

- Disables ChatSlide Module

- Disables Message Sliding

- Disables Focus Module

- Disables Blur
